```

3. Instala dependencias (DXVK, VCRedist, etc)

6. Crea launcher
7. ¡Listo!

TODO EN ~5 MINUTOS
```

## 🚀 CÓMO USAR

```bash
# En Windows:
python nexusos-windows-installer.py

```
